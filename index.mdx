    Quick Start
  </h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">
    Get up and running with Kamotake in minutes
  </p>
  
  <Steps>
    <Step title="Install Kamotake">
      Clone the repository and run the installation script. Requires Ruby 3.2.2 or higher.
      
      ```bash
      git clone --recursive https://github.com/chrischriscris/kamotaketsunominokamo.git
      cd kamotaketsunominokamo
      source install.sh
      ```
    </Step>
    
    <Step title="Create a puzzle file">
      Create a text file describing your nonogram puzzle using the input format.
      
      ```txt puzzle.txt
      5 5
      2
      4
      5
      4
      2
      2
      4
      5
      4
      2
      ```
      
      <Note>
        The first line specifies columns and rows. Following lines define row constraints, then column constraints.
      </Note>
    </Step>
    
    <Step title="Solve your puzzle">
      Run Kamotake with your puzzle file to get the solution.
      
      ```bash
      kamotake puzzle.txt -c
      ```
      
      The solver will translate the puzzle to CNF format, run the SAT solver, and display the solution as a visual grid.
    </Step>
  </Steps>
</div>
