# Quickstart Guide

This guide will help you solve your first nonogram puzzle using Kamotake's CLI.

## Your First Puzzle

Let's create and solve a simple 5x1 nonogram puzzle.

<Steps>
  <Step title="Create a Puzzle File">
    Create a file named `simple.txt` with the following content:
    
    ```txt simple.txt
    5 1
    3
    0
    1
    1
    1
    0
    ```
    
    **Format explanation:**
    - Line 1: `5 1` = 5 columns, 1 row
    - Line 2: `3` = row constraint (3 consecutive filled cells)
    - Lines 3-7: Column constraints (0 means empty column, 1 means one filled cell)
    
    This puzzle has one row with 3 consecutive filled cells.
  </Step>

  <Step title="Solve the Puzzle">
    Run Kamotake with your puzzle file:
    
    ```bash
    kamotake simple.txt
    ```
    
    You'll see the solving process:
    
    ```
    Wait while we're painting your nonogram in CNF 游녲...
    Picture completed in 0.023 seconds! 游꿦 (DIMACS translation generated in .tmp_kamotake/simple.cnf)
    Cleaning DIMACS and SAT solution files... 游뛆 (You can keep them with the -d flag)
    
    #  3
       .  餃쩚잺  餃쩚잺  餃쩚잺  .
    ```
    
    The solution shows the filled cells (餃쩚잺) forming a group of 3 in the middle.
  </Step>
</Steps>

## Command-Line Flags

Kamotake supports several optional flags:

### Keep Debug Files (`-d`)

By default, Kamotake removes temporary CNF and solution files. Use `-d` to keep them:

```bash
kamotake simple.txt -d
```

This creates a `.tmp_kamotake/` directory with:
- `simple.cnf` - The CNF clauses
- `simple_header.cnf` - The DIMACS header with variable/clause count
- `simple.sol` - The SAT solver output

<Info>
Debug files are useful for understanding the SAT encoding or debugging complex puzzles.
</Info>

### Compact Output (`-c`)

For a cleaner, constraint-free view:

```bash
kamotake simple.txt -c
```

Output:
```
  餃쩚잺餃쩚잺餃쩚잺餃쩚잺餃쩚잺餃쩚잺
```

### Combining Flags

You can use both flags together:

```bash
kamotake simple.txt -d -c
```

## More Complex Example

Let's try a larger 5x5 puzzle. Create `heart.txt`:

```txt heart.txt
5 5
3
5
3
1
1
1 1
2 1
5
2 1
1 1
```

**Format breakdown:**
- `5 5` = 5 columns, 5 rows
- Next 5 lines = row constraints
- Last 5 lines = column constraints

Solve it:

```bash
kamotake heart.txt
```

Expected output:

```
########  1  1  2  2  1
########  1  1  5  1  1
      3  .  餃쩚잺  餃쩚잺  餃쩚잺  .
      5  餃쩚잺  餃쩚잺  餃쩚잺  餃쩚잺  餃쩚잺
      3  .  餃쩚잺  餃쩚잺  餃쩚잺  .
      1  .  .  餃쩚잺  .  .
      1  .  .  餃쩚잺  .  .
```

## Using Example Puzzles

Kamotake includes example puzzles in the repository's `data/` directory:

```bash
# Navigate to the repository
cd kamotaketsunominokamo

# Solve an example puzzle
kamotake data/test0.txt -c
```

Available examples:
- `test0.txt` - 25x30 puzzle
- `test1.txt` - 53x53 complex puzzle
- `test2.txt` - 5x1 simple puzzle
- Additional puzzles: `test3.txt` through `test7.txt`

## Understanding the Output

### Standard Output Format

```
####  <column constraints>
<row> <solution cells>
```

- `#` marks the constraint area
- Numbers show the constraints for each row/column
- `餃쩚잺` represents filled cells
- `.` represents empty cells

### Solution Process

1. **CNF Translation**: Kamotake generates Boolean clauses representing all valid cell configurations
2. **SAT Solving**: The Glucose solver finds a satisfying assignment
3. **Extraction**: The solution is converted back to a grid

<Note>
If a puzzle has no solution, Kamotake will report "No solution found 游붏"
</Note>

## Puzzle File Format Reference

The input format is:

```txt
<columns> <rows>
<row_1_constraints>
<row_2_constraints>
...
<row_N_constraints>
<col_1_constraints>
<col_2_constraints>
...
<col_M_constraints>
```

**Rules:**
- Each constraint line contains space-separated numbers
- Each number represents a group of consecutive filled cells
- Groups must have at least one empty cell between them
- Use `0` for rows/columns with no filled cells

### Example: 3x3 Grid

```txt
3 3
1 1
3
1 1
1 1
3
1 1
```

This creates:
- 3 columns, 3 rows
- Row 1: two groups of 1 cell each
- Row 2: one group of 3 cells
- Row 3: two groups of 1 cell each
- Columns follow the same pattern
