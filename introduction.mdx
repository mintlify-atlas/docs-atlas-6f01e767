# What is Kamotake?

Kamotake (カモタカツノミノカモ) is a SAT-based nonogram solver that transforms logic puzzles into Boolean satisfiability problems. It provides both a command-line interface for quick solving and a web interface for interactive puzzle solving.

## What are Nonograms?

Nonograms, also known as picross or griddlers, are logic puzzles where you fill in cells on a grid according to numerical constraints. Each number indicates a consecutive group of filled cells in that row or column.

For example, a row with constraints `[3, 1]` means:
- A group of 3 consecutive filled cells
- At least one empty cell
- A group of 1 filled cell

Solving these puzzles reveals a hidden picture.

## How Kamotake Works

Kamotake uses a sophisticated approach to solve nonograms:

1. **SAT Translation**: Converts nonogram constraints into CNF (Conjunctive Normal Form) clauses
2. **Tseitin Transformation**: Uses auxiliary variables to efficiently encode constraints
3. **Glucose SAT Solver**: Leverages a modern SAT solver for optimal performance
4. **Solution Extraction**: Translates the SAT solution back into the solved grid

## Output Modes

Kamotake offers two visualization modes:

### Standard Mode
Displays the full puzzle with row and column constraints alongside the solution:

```
####  3  5  7  9 11
   3  .  .  .  ◼️  ◼️
   5  .  ◼️  ◼️  ◼️  ◼️
   7  ◼️  ◼️  ◼️  ◼️  ◼️
```

### Compact Mode (`-c` flag)
Shows only the solved puzzle for a cleaner view:

```
      ◼️◼️◼️◼️
  ◼️◼️◼️◼️◼️◼️
◼️◼️◼️◼️◼️◼️◼️◼️
```

## Use Cases

- **Puzzle Solving**: Quickly solve nonogram puzzles of any size
- **Algorithm Research**: Study SAT-based approaches to constraint satisfaction
- **Educational Tool**: Learn about Boolean logic and CNF transformations
- **Validation**: Verify manually solved puzzles or check puzzle uniqueness
